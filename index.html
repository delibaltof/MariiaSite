02

Языки и технологии (frontend)

Технология Для чего Когда выбирать Порог входа
HTML + CSS + JS Базовый стек. Любой сайт на этом работает «под капотом» Лендинги, простые сайты, прототипы Низкий
React Библиотека UI. Компонентный подход, SPA-приложения Веб-приложения, дашборды, SaaS Средний
Next.js React-фреймворк. SSR, SSG, файловый роутинг, API routes Почти всё — де-факто стандарт 2025-26 Средний
Astro Статика + частичная гидрация. Максимальная скорость Блоги, лендинги, SEO-сайты Низкий
Vue / Nuxt Альтернатива React. Более мягкая кривая обучения Корпоративные проекты, командная разработка Средний
Svelte / SvelteKit Компиляция в чистый JS. Минимум runtime-кода Перформативные сайты, нишевые проекты Средний
03

Backend (серверная часть)

Node.js
JS/TypeScript
Единый язык на фронте и бэке. Экосистема npm. Фреймворки: Express, Fastify, Hono.
Python
FastAPI / Django
Идеален при работе с AI/ML. FastAPI — быстрый REST API, Django — полный стек с ORM и админкой.
PHP
Laravel
До сих пор жив. Laravel — зрелый MVC-фреймворк. WordPress построен на PHP.
Go
Golang
Высоконагруженные API. Компилируется в бинарник, работает молниеносно.
BaaS
Supabase / Firebase
Backend как сервис. Без написания серверного кода — база данных, авторизация, хранилище.
Serverless
Vercel / Cloudflare
Edge-функции. Пишешь функцию — платформа сама масштабирует. Идеально с Next.js.

Редактирование контента (CMS)

CMS Тип Плюсы Для кого
WordPress Традиционная Огромная экосистема плагинов, визуальный редактор, хостинг везде Блоги, корп. сайты, не-разработчики
Sanity Headless Гибкая схема данных, GROQ-запросы, real-time Разработчики + нетехнические редакторы

Payload CMS Self-hosted TypeScript, Next.js интеграция, code-first подход JS-разработчики, современные проекты
Ghost Специализированная Создан для блогов и медиа, встроенная монетизация, красивый редактор Авторы, медиа-проекты
05

Публикация и хостинг

★ Рекомендовано 2026
Vercel
Push в GitHub → сайт онлайн за секунды. CDN по всему миру. Идеально для Next.js. Бесплатный тариф щедрый.
Serverless CDN CI/CD
Статика / JAMstack
Cloudflare Pages
Быстрее Vercel на статике. Workers на edge. Бесплатный тариф без ограничений по bandwidth.
Edge Free
VPS / Полный контроль
DigitalOcean / Hetzner
Виртуальный сервер. Полный root-доступ. Docker + Nginx. Нужны DevOps-знания. Hetzner — дёшево и надёжно.
Docker Nginx
06

Администрирование

Задача Инструмент Примечание
Мониторинг ошибок Sentry Ловит JS-ошибки в проде, трекинг перфоманса
Аналитика Plausible / Google Analytics 4 Plausible — без cookies, GDPR-friendly. GA4 — мощнее, сложнее
База данных PostgreSQL + Neon/Supabase Postgres — стандарт. Neon/Supabase — managed-хостинг с бесплатным тарифом
Авторизация Clerk / Auth.js / Supabase Auth Clerk — plug-and-play, Auth.js — open-source, гибко
Домен и DNS Cloudflare + любой регистратор DNS через Cloudflare — быстро, защита от DDoS, бесплатно
SSL-сертификат Автоматически Vercel, Cloudflare, Netlify выдают HTTPS автоматически — вручную настраивать не нужно
Деплой / CI-CD GitHub Actions + Vercel Пуш в main → автодеплой. Бесплатно для public репозиториев
Email-транзакции Resend / Postmark Resend — лучший DX в 2025-26, нативная интеграция с React Email

Рекомендуемый современный стек

// Универсальный старт 2026

Frontend:Next.js 15 + TypeScript + Tailwind CSS
Backend:Next.js API Routes или Hono (если отдельный бэк)
БД:PostgreSQL через Supabase или Neon
CMS:Sanity (если нужен редактор) или MDX-файлы (если технически)
Auth:Clerk (быстро) или Auth.js (контроль)
Хостинг:Vercel (деплой) + Cloudflare (домен/DNS)
Аналитика:Plausible или Vercel Analytics
GitHub → Push → Vercel CI → Prod → Cloudflare DNS → yourdomain.com ✓
